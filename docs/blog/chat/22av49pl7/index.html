自己紹介!

2026-06-04 雑談

はじめまして

初めまして。monberableのmomijiです。
0歳と2歳の子を持つ沖縄在住のハンドメイド作家です☺️

Monberableの由来

monberable（モンベラブル）は、
フランス語での愛娘の名前　と　可愛い赤ちゃんという意味のあるmonbebe
を合わせてネーミングしました。
忙しい育児の中にも可愛いを。。をモットーに制作をしています♡

ハンドメイド販売のきっかけ

小さい頃から作ることが好きで、高校は芸術家の彫刻専攻へ進み、
卒業してからは専門学校でお菓子を学んでパティシエをしていました🍰

娘の妊娠中にかぎ針編みに出会い、動画や本を見ながら編んでました🧶
自分で考えたものが形にできるようになった頃、
皆さんにも『私の作った作品で少しでも幸せになれないかな』と思うようになり、
販売することにしました☺️

最後に

ブログでは編み物の楽しさ、試作中の出来事、作品に対する想いを書けたらと思っています＾＾

最後まで読んでいただきありがとうございます。
これから、よろしくお願いいたします🍁
